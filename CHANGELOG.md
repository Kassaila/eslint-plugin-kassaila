# Changelog

## 0.1.0 (2026-03-26)

### Features

- **jsdoc-comment-style** — enforce JSDoc-style block comments, forbid inline comments, validate TODO format
- **prefer-arrow-without-this** — convert functions to arrows when `this` is unused
- **switch-case-braces** — require braces around case/default clauses
- Recommended config with all rules enabled
- Dual ESM/CJS distribution
